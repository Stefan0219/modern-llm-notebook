Modern LLM Notebook

21 个 Jupyter Notebook，手写核心算法，从零理解大语言模型的完整技术栈。直觉理解 → 手算验证 → 代码实现 → 实验观察。

21 Notebooks 5 Parts 20+ Papers PyTorch 2.0+
Quick Start Learning Map Part 1 Part 2 Part 3 Part 4 Part 5 Papers

Quick Start

# 克隆仓库
git clone https://github.com/sanbuphy/modern-llm-notebook.git
cd modern-llm-notebook

# 安装依赖
pip install -r requirements.txt

# 从第一个 Notebook 开始
jupyter notebook notebooks/part1-foundation/01-tokenizer-basics.ipynb

要求: Python 3.9+, PyTorch 2.0+, 16GB RAM。每个 Notebook 自包含，可独立运行。
